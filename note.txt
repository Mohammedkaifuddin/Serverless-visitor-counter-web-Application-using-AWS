### ❓ What is serverless?

👉 No need to manage servers; AWS handles infrastructure automatically.

---

### ❓ Why did you use Lambda?

👉 To run backend code without managing servers.

---

### ❓ Why DynamoDB?

👉 It’s a fast, scalable NoSQL database for real-time data.

---

### ❓ What is API Gateway?

👉 It acts as a bridge between frontend and backend.

---

### ❓ Why S3?

👉 To host static website files like HTML and JS.

👉 DynamoDB handles concurrent updates, and Lambda scales automatically.
